# איפה להעלות את תמונת הבסיס - הגנת זומבים

## מיקום הקובץ

**העלה את התמונה לנתיב הבא:**

```
TolePlay/public/base.png
```

כלומר: בתוך תיקיית הפרויקט שלך, בתיקייה `public`, קובץ בשם `base.png`.

## המלצות
- **שם הקובץ:** חייב להיות בדיוק `base.png`
- **גודל מומלץ:** 100x100 עד 256x256 פיקסלים
- **פורמטים:** PNG, JPG, WebP

המשחק יטען את התמונה אוטומטית ויציג אותה במקום הבסיס המצויר. אם הקובץ לא קיים, יוצג בסיס מצויר כברירת מחדל.
